# Sistema de Orçamentos para Empresas

Este é um sistema completo para gestão de orçamentos empresariais, desenvolvido para aumentar a eficiência e reduzir custos.

## Funcionalidades

- Cálculos automatizados de orçamentos
- Geração de relatórios detalhados
- Colaboração em equipe
- Armazenamento na nuvem
- Exportação para múltiplos formatos
- Segurança avançada

## Como usar

1. Acesse o sistema através do link: [Sistema de Orçamentos](https://script.google.com/macros/s/AKfycbyEbqiYpxoAvJyxhOUvf3xsbdrZhiGpPhi1aGOeA_NLKGlLhuXud6RcMO7Eiy4DgBaz/exec)
2. Configure os dados da sua empresa
3. Comece a criar orçamentos

- Google Apps Script
